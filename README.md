# Інструкція з встановлення та налаштування програми

Вітаємо! Ця інструкція допоможе вам швидко налаштувати програму для роботи з тестами.

## Крок 1: Розміщення програми

1. Створіть нову папку в будь-якому зручному для вас місці на комп'ютері (наприклад, на диску `D:\` або в папці "Документи"). Назвіть її, наприклад, `TestSystem`.
2. Перемістіть файл програми (`.exe`) у цю папку.

## Крок 2: Створення необхідних папок

Щоб програма працювала коректно, їй потрібні дві папки для зберігання даних.

1. Відкрийте папку, куди ви поклали `.exe` файл.
2. Створіть поруч із ним дві нові папки:
   - Одна для зберігання самих тестів (наприклад, назвіть її `tests`).
   - Друга для збереження результатів (наприклад, назвіть її `results`).

**Структура папки має виглядати так:**


Ваша_Папка/
├── Program.exe
├── tests/
└── results/

## Крок 3: Створення ярлика (для зручності)

Щоб щоразу не шукати папку з програмою:

   Натисніть правою кнопкою миші на файл програми (.exe).

   У меню виберіть "Надіслати" -> "Робочий стіл (створити ярлик)". Тепер ви можете запускати програму прямо з робочого столу.

Крок 4: Налаштування шляхів у програмі

   Запустіть програму через створений ярлик або .exe файл.

   Перейдіть у меню "Налаштування" (Settings).

   У відповідних полях вкажіть шляхи до створених вами папок:

   У полі для Тестів виберіть папку tests.

   У полі для Результатів виберіть папку results.

   Збережіть налаштування.

Готово! Програма налаштована і готова до роботи.
